How to Use in Your README

To use the counter in your GitHub profile README, add the following code:

![Nixie Tube Counter](http://yourserver.com/simple.php?username=yourusername)

For testing locally:

![Nixie Tube Counter](http://localhost:8080/simple.php?username=testuser)

Customization Options

Change the number of digits (default is 6):

![Nixie Tube Counter](http://localhost:8080/simple.php?username=testuser&digits=8)

Set a starting base count:

![Nixie Tube Counter](http://localhost:8080/simple.php?username=testuser&base=1000)
